# ✅ ToDo Flask App

Welcome to the **ToDo Flask App** – a clean and simple task management tool built using Python and Flask.  
It allows you to create, update, and delete tasks, helping you stay organized and productive! 🧠🗂️

## 🔧 Tech Stack

- 💻 Python
- 🌐 Flask (Web Framework)
- 🎨 HTML5, CSS3 (Frontend)
- 🗃️ SQLite (Database)

---

## 🛠️ Features

✨ **User-Friendly Interface** – Add, update, and delete tasks effortlessly  
⏰ **Deadline Support** – Keep track of due dates  
✅ **Completion Status** – Mark tasks as done  
💾 **Persistent Storage** – Tasks are saved in SQLite  
📱 **Responsive Design** – Works on desktops & mobile devices
